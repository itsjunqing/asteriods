# asteriods-gameplay
A simple asteriods game built with [Typescript](https://github.com/microsoft/TypeScript), [jQuery JavaScript library](https://github.com/jquery/jquery) and [RxJx Observables](https://rxjs-dev.firebaseapp.com/guide/observable).

## How to Play
Head to https://itsjunqing.github.io/asteriods-gameplay/<br>
Full instructions are described in the link above. Note that this game is only compatible with Chrome browser. If you encounter issues when using other browsers, please switch to Google Chrome browser.
